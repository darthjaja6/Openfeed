# Local Web Feed Example

This example reads from TikTok and pushes cards to OpenFeed's built-in
localhost web server via `consumer_type: local_web`. It is the Quickstart path
and does not require a Ticlawk account.

Copy `openfeed.yaml`, `.env.local.example`, and `run-openfeed` to your instance
root. Install Browser Bridge, log in to TikTok in Google Chrome, then run:

```bash
./run-openfeed start
```
